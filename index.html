Demande de suppression de compte

Bienvenue sur la page de suppression de compte pour Inter Cook.

Procédure de suppression de compte

Pour supprimer votre compte et les données associées, veuillez suivre les étapes suivantes :

  1. Envoyez un e-mail à notre support à intercook2024@gmail.com en indiquant l'adresse e-mail associée à votre compte.
  2. Notre équipe confirmera votre demande de suppression sous 48 heures.
  3. Une fois votre demande confirmée, votre compte et toutes les données associées seront supprimés dans les 14 jours ouvrables.

Données supprimées ou conservées

Les types de données supprimés incluent :

  • Données de profil (nom, adresse e-mail, etc.).
  • Historique des activités et préférences utilisateur.

Notez que certaines données peuvent être conservées temporairement pour respecter des obligations légales (comme les données de facturation) avant suppression complète.

Durée de conservation

Les données sont conservées jusqu'à 14 jours après la confirmation de votre demande. Passé ce délai, toutes les informations seront supprimées de façon permanente.

Pour toute question supplémentaire, contactez-nous à intercook2024@gmail.com.
